This is the Demo System for the Intel-NTU DVC project.

Requirement:
- opencv library
- mplayer
- x.264
- h264 JM reference codec

Compilation:
- using 'make all' to compile everything.

Run:
- cd to the codec_h264 directory
- run 'sh encode.sh' for encoder side computer
- run 'sh decode.sh <encoder IP>' for decoder side computer
